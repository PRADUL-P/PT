1. Introduction & Overview

Welcome to TendonFlow Documentation. This user guide contains everything you need to build, modify, verify, and export post-tensioned (PT) concrete slabs. TendonFlow computes continuous tendon drapes using double-quadratic parabolas, and solves friction and anchor slip stress profiles along the strand trajectories.

Figure 1.1: Visual schematic explaining how editing specific parameter fields affects the PT geometry.

Interactive Designer Workspace

Parameters specified in the sidebar panel are linked directly to our real-time math engine and SVG visualizer. Changes made in the interface are immediately verified against structural cover limits, load balancing ranges, and curvature checks.

2. Slab Geometry Parameters

Slab bounds define the boundaries for concrete self-weight and tendon drape calculations. Adjusting these values scales the design coordinate canvas:

Number of Spans 1 - 3

Sets how many continuous structural spans are modeled longitudinally. To change: Use the "Number of Spans" dropdown. Length controls populate automatically.

Number of Column Rows 1 - 4

Specifies how many transverse column rows are positioned along the design grid in plan layout (e.g. 1 Row in center, or 3 Rows for Left, Mid, Right grid lines). To change: Use the "Number of Column Rows" dropdown.

Revit: N/A

Measurement Unit cm / mm

Toggles the display and input units for clearances, covers, heights, and anchor sets. To change: Select "Centimeters (cm)" or "Millimeters (mm)" from the "Measurement Unit" dropdown.

Revit: Standardized to Metric (mm)

Slab Width 2 - 100 m

The design tributary width. Determines the total concrete self-weight to balance. To change: Enter the width value in the width numeric field.

The unit weight of the concrete, used to calculate the gravity self-weight load ($w = \text{density} \times \text{thickness} \times \text{width}$). To change: Enter value in the "Density" field.

Slab Corner Elevations (Bottom Z)

When the PT slab is sloped or warped, you can specify individual Z-height offsets at the four corners of the concrete deck. These are bilinearly interpolated across the slab width and length. Enabling this dynamically wraps the 3D concrete mesh and offsets the elevation profiles of all longitudinal (X) and transverse (Y) tendons in 3D CAD exports.

4. Tendon & Prestressing Parameters

These coefficients specify the friction losses as strands weave through duct paths. Friction reduces effective prestressing forces along the spans:

P(x) = P_0 · e^(-(μθ + kx))

X & Y Jacking Force 100 - 5000 kN

The jacking tension (P0) at the anchors. To change: Enter force values in "X Jacking Force" and "Y Jacking Force".

Revit: TF_TendonForce (Number)

Jacking End Left / Right / Both

Specifies which ends of the continuous tendons are jacked. Jacking both ends reduces overall friction loss peak. To change: Select from the "Jacking End" dropdown.

Revit: N/A

Anchor Set 0 - 1.5 cm

Slippage distance (draw-in) during anchorage lockoff. To change: Type the value in the "Anchor Set" field.

Revit: N/A

Curvature Coeff (μ) 0.05 - 0.30

Friction coefficient representing steel sliding inside the duct. To change: Enter value in the "Curvature Coeff (μ)" field.

Revit: N/A

Wobble Coeff (k) 0.0005 - 0.005

Accidental wobble misalignment factor per meter. To change: Adjust the "Wobble Coeff (k, rad/m)" field.

Revit: N/A

Tendon X & Y Spacing 0.5 - 5.0 m

The average spacing between adjacent strands in the grid, used for calculating balanced pressure fields. To change: Adjust "X Spacing" and "Y Spacing".

Revit: N/A

Duct Outer Diameter 1.0 - 15.0 cm

The outer thickness (size) of the post-tensioning duct. Increases in duct diameter reduce the available vertical drape height. To change: Enter the size in "Duct Outer Diameter".

5. Trajectory & Profile Heights Editing

Adjusting the vertical profile drape is straightforward. In the **Elevation Profile** tab, look at the active blue node dots along the tendon curve:

  • Support Nodes (High Points): Positioned directly above column lines. Drag up/down to modify support high points.
  • Span Nodes (Low Points): Positioned at midspan. Drag up/down to modify low point drape; drag left/right to shift the peak position.

Tendon Profile Heights Sidebar Parameters

You can also edit the profile heights and low-point positions numerically inside the sidebar. The fields are defined as follows:

Support Clearances From Top

Support S0, S1, S2 Clearance: The distance measured from the top face of the slab down to the tendon centerline. Decreasing this value pushes the tendon closer to the top face (increasing the tendon elevation height over support columns).

Revit: TF_Support_0 to _3 (Length, measured above bottom of slab)

Mid-Span Low Points From Bottom

Height (Ht): The vertical distance measured from the bottom face of the slab up to the lowest point of the tendon centerline.
Position (Pos x): The horizontal position of the lowest point, measured in meters from the left support of that respective span.

Revit: TF_LowPoint_1 to _3 (Length, measured above bottom of slab)

Support Slope Angles Entry / Exit

Shows the slope angle (in degrees) of the tendon path as it enters and exits each column support. These are updated in real-time (e.g. 10.6° or 8.3° | 5.3°) to help verify structural bending checks.

6. Perpendicular (Transverse) Tendons

Perpendicular tendons run in the transverse (Y) direction across the slab, crossing the main longitudinal (X) tendons. They are distributed in groups or bands near column support lines to satisfy punching shear and distribution reinforcement checks.

Figure 6.1: Plan diagram of a perpendicular tendon set showing support alignment, count, offset, and spacing parameters.

Number of Tendon Sets 0 - 10

Sets how many unique groups of perpendicular tendons are modeled in the slab. To change: Adjust the number input under "Number of Tendon Sets".

Support Location S0, S1, S2, etc.

The column support index where the set is anchored or located. To change: Select from the "Support" dropdown inside the tendon set card.

Band Direction Left / Right

Specifies whether the tendon group lies on the left side or the right side of the support centerline. To change: Select "Left" or "Right" from the "Direction" dropdown.

Tendon Count 0 - 50

The total number of perpendicular strands in this group. To change: Modify the "Count" numeric input.

Spacing min 5 cm / 50 mm

The center-to-center spacing between consecutive perpendicular tendons in this group. To change: Adjust the "Spacing" field.

Offset min 0

The horizontal clearance distance from the column support line to the first perpendicular tendon. To change: Adjust the "Offset" field.

Elevation Height 0 - Slab Thickness

The vertical elevation (measured from the bottom of the slab to the tendon centerline) where this perpendicular group runs. Used for 3D clash checks. To change: Type the value in the "Height" field.

3D Clash Check & Verification

TendonFlow dynamically compares the vertical height of these perpendicular tendons with the longitudinal tendon profiles. If the duct sizes overlap, a warning is instantly flagged in the "Design Verification & Checks" panel to prevent on-site construction clashes.

7. 2D Plan Layout Parameters

The 2D Plan view is used to lay out the tendon grid in the slab plane. It shows column locations, tendon spacing, and coordinate grids:

Figure 7.1: Slab plan layout schematic showing column supports and longitudinal tendon spacing.

Define X-Tendons By Spacing / Positions

Sets how the coordinates of the longitudinal tendons are defined. Uniform Spacing spaces tendons evenly across the width. Custom Y-Positions allows placing them at custom coordinates. To change: Select option from the dropdown.

Tendon Spacing 0.5 - 5.0 m

The uniform spacing between tendons in meters. Only active when "Uniform Spacing" is selected. To change: Adjust the "Tendon Spacing" numeric input.

Y-Positions m, Comma-Sep

A comma-separated list of absolute Y-coordinates (measured from the bottom slab edge) defining custom tendon positions. To change: Type values in the "Y-Positions (m, comma-sep)" input field.

Column Coordinates Auto-populated

Table of coordinates defining where column rows intersect support lines. To change: These update dynamically based on the number of column rows and span lengths.

X-Tendon Sets & Colour Coding

When X-Tendon Sets mode is selected in the Plan Layout sidebar, each tendon set is drawn in a distinct colour (lime, purple, green, cyan, pink, orange, etc.) based on its set index. The colour coding is consistent between the Plan View and the Elevation Profile view, making it easy to identify which set you are editing. Clicking any X-tendon line in the plan view instantly selects that tendon set’s elevation profile for editing — the section tabs above the visualizer automatically switch to the correct Above Row N or Below Row N tab. For the most efficient workflow, use Split View to see both the plan and the elevation profile side by side simultaneously so that profile changes are immediately visible as you click between sets.

Tip: Click-to-Edit in Plan View

While in X-Tendon Sets mode, a single click on any tendon line in the 2D Plan Layout instantly jumps the Elevation Profile view to that set’s drape curve. If you are in Plan-only view, the interface automatically switches to Split View so you can see both panels at once.

8. View Controls & Navigation

TendonFlow provides a set of interactive view controls that let you zoom, pan, and navigate between the Elevation Profile and 2D Plan Layout views without losing context. These controls are available in every view mode.

Zoom Controls

Floating zoom control buttons appear at the top-right corner of each view panel:

Zoom In + button

Increases the zoom level of the active view by one step. Zoom range is 0.5× to 5.0×. Available individually in Elevation Profile view, 2D Plan Layout view, and in each panel of Split View.

Zoom Out – button

Decreases the zoom level of the active view by one step. Use this to get a wider overview of the full slab layout or multi-span tendon profile.

Reset Zoom ↺ button

Returns the view to the default 1:1 zoom level and resets the pan offset. Useful after navigating deep into a zoomed region.

Pan (Drag-to-Pan)

Click and drag the background of any view panel to pan freely within the canvas. Works in all view modes at any zoom level.

Split View

The Split View tab displays the Elevation Profile and the 2D Plan Layout simultaneously, side by side. This is especially useful when editing tendon drape profiles while monitoring their plan positions. Each panel has its own independent zoom and pan controls, so you can zoom into a specific tendon in the plan without affecting the elevation profile scale.

Recommended Workflow: Split View + Click-to-Edit

For the fastest tendon editing experience, switch to Split View and activate X-Tendon Sets mode in the sidebar. You can then click any tendon in the plan (left panel) and its drape profile will immediately appear in the elevation view (right panel) — ready to edit with drag handles or numeric inputs.

Tendon Colour Coding

Colour coding is applied consistently across both views to make it easy to identify which tendon set you are working on:

Plan View Colours

In X-Tendon Sets mode, each set is drawn in a unique colour (lime, purple, green, cyan, pink, orange, etc.) based on its set index. In spacing or positions modes, tendons remain red as before.

Elevation Profile Colours

The active (selected) tendon profile curve is coloured using the same set colour as shown in the plan view. Ghost profiles of other sets are drawn at low opacity with dashed lines, each in their own set colour. The legend in the top bar shows the active set colour labelled Active Tendon.

Cover Violation Override

If a concrete cover violation is detected anywhere along a tendon profile, that profile is highlighted in red regardless of its assigned set colour — ensuring violations are always clearly visible.

End Slope Angle Locking

Support exit angles can be locked to a specific value. When locked, the angle indicator shows green if the angle is within the allowed range, or red if it is outside. When unlocked, the current natural drape angle is displayed.

Corner slab elevations (BL, BR, TR, TL Z-offsets) are also accounted for when computing tendon Z coordinates in the export, ensuring correct placement in sloped or warped slab configurations.

10. Practical Use Cases & Workflows

Use Case 1: Balancing Upward Load Pressures

Continuous parabolas exert upward balanced force profiles opposing gravity downward loads. To check this:

  1. Set structural span sizes and slab width.
  2. Observe the Equivalent Load (kN/m) column in the coordinate table.
  3. Verify in the **Design Verification & Checks** panel that load balancing meets the green threshold (60% to 80% slab self-weight).

Use Case 2: Clipboard-Based Drawing (Fastest)

Draw tendon designs instantly inside an active AutoCAD session without saving separate data files:

  1. Download and load TendonFlowDraw.lsp inside AutoCAD once (using APPLOAD).
  2. In the TendonFlow export modal, select **TendonFlow Data File (.tfd)** as the format.
  3. Click **Copy LISP Data** in the modal footer.
  4. Inside AutoCAD, click the command input bar, press Ctrl + V, and hit Enter.
  5. Type DRAWELEVC (for elevation) or DRAWPLANC (for plan layout) and press Enter to draw.

11. Revit Add-in Guide & Installation

TendonFlow includes a robust pyRevit extension to bridge your web designs directly into 3D Revit models. This section outlines how to install the extension and run the direct and batch workflows.

1. Installation

To load the TendonFlow ribbon inside Revit:

  1. Click the Download Revit Extension button on the Web Designer header to download TendonFlow.extension.zip.
  2. Unzip the downloaded file.
  3. Move the extracted TendonFlow.extension folder into your local pyRevit Extensions directory (typically located at C:\Users\<YourUsername>\AppData\Roaming\pyRevit\Extensions\).
  4. In Autodesk Revit, go to the pyRevit tab on the ribbon and click Reload.
  5. A new TendonFlow tab will load with four tools: TendonFlow Designer, Tendon From Line, Batch Import, and Update Tendon.

2. Batch Import Workflow (Offline & Fast)

Import and model dozens of tendons simultaneously from a single web design file:

  1. Draw model or detail lines along your tendon plan layouts in your Revit floor view.
  2. Design your tendon elevation clearances and drapes on the TendonFlow website.
  3. Click Save Design in the header to save the .json design file.
  4. In Revit, click Batch Import on the TendonFlow tab.
  5. Select all the plan lines, click Finish in the top-left of the Revit viewport, and then click to select the host Concrete Slab Floor.
  6. Select the downloaded .json file in the file explorer.
  7. TendonFlow will automatically create the 3D swept geometries inside the slab and populate all parameters!

3. Interactive Designer Bridge Workflow

Design a specific tendon in real-time inside the Revit window:

  1. Select a single plan line and the host slab floor in Revit.
  2. Click TendonFlow Designer on the ribbon.
  3. The designer window automatically reads the slab depth, start/end locations, and span count.
  4. Adjust your drape settings and click Import to Revit inside the window.

4. Modifying and Updating Tendon Geometry

Unlike standard direct shapes, TendonFlow maintains custom parameters on modeled generic models. To edit a tendon path after modeling:

  1. Select the tendon element in Revit.
  2. In the Properties Palette, edit any clearance parameters under the Constraints group (e.g. TF_Support_0 to 3, TF_LowPoint_1 to 3, or TF_SpanLength_1 to 3).
  3. Click the Update Tendon button on the TendonFlow tab.
  4. TendonFlow will dynamically recalculate the curve and update the 3D swept duct element immediately!
